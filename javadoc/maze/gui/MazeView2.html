                      int offset,
                      boolean trimTail)
Draw a robot path onto the maze. The path will have the width of the cell walls.

Parameters:
g - Where to draw.
path - The cell to cell path to draw.
offset - Shift the path by the given amount so that you can draw multiple paths without them overlapping. A value of 0 draws to the center of the cell.
trimTail - If set to true the tail of the path will be trimmed to the location of the robot. This allows the current path to finish right on the current robot position.
